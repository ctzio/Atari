This is a greek language pack. It has some replacement system fonts for NVDI
than need to be placed at the GEMSYS folder (Monaco). 
Also you will find 6 ttf fonts that use the MSDOS greek codepage. 

You can also find 4 kbd drivers-tables. 

1. greek20c.acc - desk accessory for ST(e)'s.
2. gr-1.prg - auto folder program for TOS/Multitos up to a falcon. Doesn't work
on the 060. 
3. greekclk.kbd - greek keyboard table for clocky. 
4. greek.tbl - greek keyboard table for MiNT

If you can see the text below your installation works... :)
